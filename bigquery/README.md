# BigQuery Plugin

BigQueryのクエリ支援とベストプラクティスを提供するClaude Codeプラグイン。

## 構造

```
bigquery/
├── .claude-plugin/
│   └── plugin.json        # プラグインメタデータ
├── README.md
└── skills/
    └── bigquery/
        └── SKILL.md       # スキル定義
```

## 機能

- **クエリ支援**: SQLクエリの作成・最適化・デバッグ
- **ベストプラクティス**: パフォーマンス・コスト最適化のガイダンス

## 使用方法

BigQueryに関連する質問やタスクを行うと、スキルが自動的に適用されます。
